  - 웹 서버: 웹 서버는 정적 파일(홈페이지의 이미지 등 변하지 않는것. 단, 사용자가 올리는 이미지는 변하므로 동적 파일임. 동적 파일은 웹서버에서 관리하는 다른 스토리지에서 가져옴)을 서빙(PHP와 같은 프로그래밍 언어도 처리하기는 함.)

  - DB: 데이터 저장. (이미지는 DB에 저장하지 않음. 이미지 경로만 저장.)
