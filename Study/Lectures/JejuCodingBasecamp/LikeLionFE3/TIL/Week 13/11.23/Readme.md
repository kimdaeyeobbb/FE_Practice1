# axios (cdn 방식) 예시

```html
<!DOCTYPE html>
<html>
  <head>
    <meta charset="UTF-8" />
    <script src="https://unpkg.com/axios/dist/axios.min.js"></script>
  </head>
  <body>
    <script>
      const url =
        "https://raw.githubusercontent.com/paullabkorea/coronaVaccinationStatus/main/data/data.json";
      axios.get(url).then((response) => {
        console.log(response);
      });
    </script>
  </body>
</html>
```
