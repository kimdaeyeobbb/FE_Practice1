# 메모이제이션

- 쭉 내려가다가 이미 저장된 값은 다시 연산하지 않고 확인만하고 값을 가져옴

- 랜더링 될 때 부하가 걸리는 작업이 있다면 메모이제이션을 이용해서 기억해둠

## 예시: 피보나치 순열

- f(6)

  - 1,1,2,3,5,8

- f(5), f(4),f(3)등이 반복 계산됨
  - 비효율적
  - 이를 해결하기 위해 `메모이제이션`, `DP`등을 이용
